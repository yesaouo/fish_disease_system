## 影像標註 --- 操作 SOP

本 SOP 以目前標註頁與後端提交驗證為準。若畫面顯示與本文不同，請先回報管理者，不要用註解繞過一般必填欄位。

### 一、先判定影像類型

1. **有病灶影像**：使用病灶框標註，完成類別、外觀敘述與整體判讀後提交。
2. **無明顯病灶影像**：不要新增病灶框，使用工具列的「判定為健康」。影像會移到健康影像區。
3. **無法判讀影像**：影像模糊、遮擋嚴重、主體不完整、類別不在清單或無法合理判斷時，使用「無法使用並提交」，或在註解中寫明原因後提交。

### 二、有病灶影像標註流程

1. **檢查標註框**  
   每個病灶一個框；多處病灶請建立多個框。框應盡量貼近病灶，少含背景。框可拖曳移動、拉角縮放；不可超出影像範圍。

2. **刪除多餘框**  
   先點選目標框，再點工具列上的 Trash 圖示，或按 **Del / Backspace**。

3. **新增遺漏框**  
   點工具列上的 SquarePlus 圖示，或按 **N**，再拉出正確位置與大小。新增框會沿用上一個框的表徵類別；外觀敘述需重新選擇。

4. **確認表徵類別與外觀敘述**  
   每個非健康框都必須選擇「表徵類別」與「外觀敘述」。表徵類別改變時，外觀敘述會清空，需重新選擇。

5. **填寫病徵敘述**  
   「通俗描述」與「醫學描述」至少填一項即可；兩者都能填時仍建議都填。欄位為單行輸入邏輯，按 Enter 不會換行。

6. **填寫病徵原因與處置建議**  
   「病徵原因」必填，依可能性由高到低排序，最多 10 項且不可重複。  
   「處置建議」為選填；若填寫，依處置流程排序，最多 10 項且不可重複。

7. **註解**  
   註解用於記錄例外狀況或判斷依據，例如「影像模糊」、「疑似多重病灶」、「類別不在清單」。註解不是一般必填欄位的替代品。

8. **保存**  
   點工具列上的 Save 圖示，或按 **Ctrl + S** 暫存。保存只更新草稿，不代表任務完成。

9. **資料驗證與錯誤定位**  
   提交時若有缺漏，頁面上方會顯示警示條，並可定位到第一個錯誤。右側標註清單中有問題的框會以紅色標示。

10. **提交**  
    確認所有必填項目完成後點「提交」。提交成功後，此影像視為完成並前往下一個編號。

### 三、提交前檢查

- 每個病灶都有框，且沒有漏框、重複框或過大的背景框。
- 每個框都有表徵類別與外觀敘述。
- 病徵敘述至少填寫通俗或醫學其中一項。
- 病徵原因至少 1 項，排序合理且不重複。
- 處置建議若有填寫，排序合理且不重複。
- 無病灶影像已使用「判定為健康」，不是用空白標註提交。
- 無法判讀影像已有清楚註解或使用「無法使用並提交」。

### 四、切換、復原與離開

- 使用上方編號列的上一個、下一個或「前往」切換影像。
- 復原 / 重做可使用工具列按鈕，或按 **Ctrl + Z** / **Ctrl + Y**。
- 若存在未保存變更，離開或切換頁面時會跳出提醒；提交流程會暫時放行導頁。

### 快捷鍵小抄

| 功能 | 快捷鍵 |
| --- | --- |
| 新增框 | N |
| 刪除框 | Del / Backspace |
| 復原 | Ctrl + Z |
| 重做 | Ctrl + Y |
| 保存 | Ctrl + S |

> 提示：工具列圖示支援滑鼠懸停顯示名稱與快捷鍵；提交目前無快捷鍵。
