# 魚病標註平台 — 前端使用說明

## 你要做的事
- 看圖找出病灶
- 畫框圈選病灶位置、選擇正確表徵類別
- 選擇「外觀敘述」（下拉選單）與填寫兩句「病徵敘述」
- 補上「可能原因」與「處置建議」（依序）
- 送出或跳過任務

## 作業流程
1) 登入
- 輸入顯示名稱（中/英文，1–32 字，僅字母）與金鑰，必要時勾選「我是養殖專家」，按「開始標註」。金鑰請向管理者取得。

2) 選擇資料集
- 在「選擇資料集」頁面，點選要處理的資料集。

3) 標註一張圖（建議順序）
- 新增框：按「新增框 (N)」，拖曳移動、拉角調整大小。
- 標註內容：在右側選「表徵類別」，並從下拉選單選擇「外觀敘述」。
- 病徵敘述：各填一行「通俗描述」與「醫學描述」。
- 病徵原因：按「新增」逐項加入，依可能性排序（用箭頭）；最多 10 項、不可重複。
- 處置建議：按「新增」逐項加入，依實際處置流程排序；最多 10 項、不可重複。
- 完成後按「提交」。若暫時無法判定，選好原因後按「跳過」。
- 隨時可按「保存」或 **Ctrl + S** 暫存，避免離開時遺失變更。

## 操作要點
- 一個病灶一個框；多處病灶請建立多個框。相連病灶可用一個較大的框。
- 框盡量貼邊，少含背景。
- 表徵類別以清單既有名稱為準；不確定選最接近者，必要時把判斷依據寫在「通俗描述／醫學描述」或「註解」。
- 文字請簡短、單句、具體。

## 快捷鍵（輸入框內無效）
- N：新增框
- Delete / Backspace：刪除框（需先選中）
- Ctrl + S：保存
- Ctrl + Z：復原
- Ctrl + Y：重做

小提醒：新增/刪除/復原/重做需先點一下畫布或空白處；保存的 Ctrl + S 在任何焦點下都可用。

## 常見狀況
- 登入失敗：確認顯示名稱僅含中英文字母且有填寫金鑰；金鑰需向管理者確認。
- 沒有任務可派：稍後再試或改選其他資料集。
- 影像載入失敗：重新整理；若持續發生，請回報資料集名稱。
- 顯示已被他人更新：系統會自動派發下一張，直接繼續。
- 可能遇到覆核：有些影像會再次分派做覆核，您照一般流程完成即可。

## 提交前 10 秒檢查
- 每個框都有正確類別、位置合理，且已選擇外觀敘述。
- 病徵敘述已填通俗 1 句 + 醫學 1 句。
- 病徵原因與處置建議：順序合理、無重複、各不超過 10 項。

## 求助
需要新增/調整類別、資料集或遇到問題，請聯絡管理者。回報請附：顯示名稱、資料集名稱、問題描述與截圖（若可）。
